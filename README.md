💊 SafeMeds

SafeMeds is a full-stack web application designed to ensure safe and reliable medicine usage. It helps users access verified medicine information, interact with healthcare support, and avoid counterfeit or unsafe drugs.

🚀 Features
🔐 User Authentication (Login / Signup)
💊 Medicine Information System
🤖 Chat Support System
🌐 Full-stack integration (Frontend + Backend)
🔒 Secure API with authentication

🛠️ Tech Stack
Frontend
React (Vite)
HTML, CSS, JavaScript
Backend
Node.js
Express.js
Database
MongoDB
