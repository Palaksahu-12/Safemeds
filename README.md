💊 SafeMeds

SafeMeds is a full-stack web application designed to ensure safe and reliable medicine usage. It helps users access verified medicine information, interact with healthcare support, and avoid counterfeit or unsafe drugs.

🚀 Features

1.🔐 User Authentication (Login / Signup)
2.💊 Medicine Information System
3.🤖 Chat Support System
4.🌐 Full-stack integration (Frontend + Backend)
5.🔒 Secure API with authentication

🛠️ Tech Stack:

1.Frontend :
React (Vite)
HTML, CSS, JavaScript
2.Backend:
Node.js
Express.js
3.Database:
MongoDB
